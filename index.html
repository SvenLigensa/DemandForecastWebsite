Demand Forecast

Why Demand Forecast?
Demand forecasting tries to predict customer demand. It is a requirement for further planning like production planning. This can be done in many different ways including time series projection methods. On this website you can play with three simple forecasting methods: the Moving Average, the Single and the Double Exponential Smoothing. More information on each of the methods can be found in the corresponding tabs. Enter your own data or use predefined data. Try different parameters to see how well the predictions represent the data.

Moving Average
The Moving Average represents the unweighted mean of the demands of the previous N periods. If the value for N is too small nervous variations can occur. If it is too big the forecast will only react slowly to changes of the data as new data-points have less impact. It is used for data with a constant level.

Single Exponential Smoothing
The Single Exponential Smoothing weights the demands of recent periods heavier than more distant ones. The new data-point is weighted with α (alpha), the forecast of the preceding period is weighted with (1-α). α is called the smoothing factor. The forecast is initialized as the first data-point. It is used for data with a constant level.

Double Exponential Smoothing
The forecast of Double Exponential Smoothing not only includes the level of the previous data points, but also the slope between those points. α is called the data smoothing factor, β is called the trend smoothing factor. The base value is computed by weighting the new data-point with α and the forecast of the preceding period with (1-α). The slope prediction is computed by weighting the slope-prediction between the last two base values with β and the slope prediction of the last period with (1-β). The forecast is initialized as the first data-point, the slope-predicion is initialized as the slope between the first two data-points. It is used for data with a trend.
